This game was built for #procjam 2014. Procjam is a game jam about procedural generation.

This is a Roguelike in which the levels are generated procedurally. The code generates drones in order to visualise this procedural process.

You can view the source code here.

Written by Christopher Waite (@chrismwaite).

Symbols and their meaning:

  • @ You, the hero.
  • # Walls.
  • . Floors.
  • ~ Water and other liquids.
  • = The exit. Enter this and the level will regenerate.
  • b Bats, Eek!
  • s Snakes, Eek!
